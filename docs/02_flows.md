# 02. Flujos
## Consumir un componente
1. Dev identifica que necesita una sección (ej: página 404)
2. Revisa el catálogo de la librería
3. Copia el componente al proyecto destino
4. Personaliza via props (textos, colores, callbacks)
5. Integra en su layout
## Agregar un componente
1. Maintainer identifica un pattern repetido
2. Abstrae en componente genérico con props
3. Lo agrega a la librería con documentación
4. Commit + push
