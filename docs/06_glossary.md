# 06. Glosario
| Término | Definición |
|---------|-----------|
| Section Block | Componente de UI que representa una sección completa de página |
| Props | Parámetros de configuración del componente |
| Slot | Área del componente donde se inyecta contenido custom |
| Composition Pattern | Patrón de componentes que se componen entre sí |
| Theme Token | Variable de diseño (color, spacing, etc.) |
